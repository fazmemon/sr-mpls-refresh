# SR-MPLS Refresh

Single-file reveal.js deck for an MPLS/TE designer refreshing to state-of-the-art SR-MPLS, EVPN, DC fabrics, and cloud networking.

Open `index.html` in any browser, or read it as a hosted slide deck:
**https://fazmemon.github.io/sr-mpls-refresh/**

## Sections

1. SR-MPLS protocol — segments, SRGB, flex-algo
2. IGP + BGP extensions for SR — IS-IS, OSPF, BGP-LS, BGP-LU+Prefix-SID
3. TI-LFA — evolution from RSVP-TE FRR → LFA → RLFA → TI-LFA
4. Flex-Algo deep dive — FAD, ASLA, per-algo SIDs, pitfalls
5. ECMP / UCMP — failure modes, with/without SR
6. Physical topology economics
7. Line cards — Cisco / Nokia / Juniper
8. Inter-AS + Egress Peer Engineering
9. Coverage matrix (PCE strategies, failure budget, UCMP triggers)
10. Two reference cases — MPLS shop vs IP-only
11. Labs — ipspace SR Workshop via Codespaces
12. DC architecture — CLOS, HFT, AI training, hybrid
13. Cloud primer — AWS / Azure / GCP
14. SRv6 appendix
15. Resources
