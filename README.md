# SR-MPLS Refresh

Single-file reveal.js deck for an MPLS/TE designer refreshing to state-of-the-art SR-MPLS, EVPN, DC fabrics, and cloud networking.

Open `index.html` in any browser, or read it as a hosted slide deck:
**https://fazmemon.github.io/sr-mpls-refresh/**

## Sections

1. SR-MPLS protocol — segments, SRGB, flex-algo
2. IS-IS wire detail — what's actually flooded
3. ECMP / UCMP — failure modes, with/without SR
4. Physical topology economics
5. Line cards — Cisco / Nokia / Juniper
6. Inter-AS + Egress Peer Engineering
7. Coverage matrix (PCE strategies, failure budget, UCMP triggers)
8. Two reference cases — MPLS shop vs IP-only
9. Labs — ipspace SR Workshop via Codespaces
10. DC architecture — CLOS, HFT, AI training, hybrid
11. Cloud primer — AWS / Azure / GCP
12. SRv6 appendix
13. Resources
